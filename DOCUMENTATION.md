## 🚀 Як працює маршрутизація / How Routing Works

### Сценарій 1: На вашому комп'ютері (Локально)
```
1. Ви відкриваєте http://localhost:8000/
2. index.html перевіряє мову браузера
3. JavaScript перенаправляє на /ua/ або /pl/ або /en/
4. Браузер завантажує правильну версію
```

### Сценарій 2: На сервері з Apache
```
1. Користувач заходить на https://yourdomain.com/
2. .htaccess перевіряє заголовок Accept-Language
3. Сервер автоматично перенаправляє на /ua/, /pl/ або /en/
4. HTML, CSS та JS завантажуються
```

### Сценарій 3: Ручна мовна вибір
```
Користувач натискає на мовний перемикач в навігації:
UA | PL | EN
Переходить на обрану мову одним кліком
```

## 🎨 Мовний перемикач / Language Switcher

Знаходиться в навігації на кожній сторінці:

```html
<div class="lang-switcher">
  <a href="/ua/" class="lang-link active">UA</a>
  <a href="/pl/" class="lang-link">PL</a>
  <a href="/en/" class="lang-link">EN</a>
</div>
```

**CSS стилі:**
- `lang-switcher` - контейнер перемикача
- `lang-link` - звичайне посилання
- `lang-link.active` - активна мова (підкреслена, кораловий колір)

## 📦 Розгортання на сервер / Deploying to Server

### Вимоги:
